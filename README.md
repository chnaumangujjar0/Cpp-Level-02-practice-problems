# C++ Practice Programs 🚀

This repository contains a collection of **C++ practice programs** that I implemented while learning and strengthening my programming fundamentals.
The goal of this repository is to build a strong understanding of **core programming concepts such as arrays, recursion, loops, and mathematical problem solving**.

Each program focuses on solving a specific problem and helps practice different programming techniques in C++.

---


## 🧠 Concepts Covered

This repository includes programs related to:

* Array operations (max, min, sorting, reversing)
* Finding the second largest element
* Merging arrays
* Removing duplicate elements
* Mathematical algorithms
* Armstrong numbers
* Fibonacci sequence
* GCD and LCM
* Recursion
* Simple calculator implementation

These exercises are designed to improve **problem-solving skills and programming logic**.
